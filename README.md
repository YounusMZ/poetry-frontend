# Poetry

## Description

Frontend for [poetry-backend](https://github.com/YounusMZ/poetry-backend). Webapp to view random poems for the fun, search for poems by title and collect your favourites.

## Demo
The live site can be found on the [backend repo](https://github.com/YounusMZ/poetry-backend). Screenshots and more details can be found there as well. If you're looking to run the app, be sure to head there. If you're looking to work on the frontend, you're at the right place!

## Features
- Shows random poems on the homepage. Explore every day
- Migrate poems from CSV and JSON to local SQLITE database
- Search for poems by title
- Boomark your favourite poems
- Responsive UI - Use it on your phone!

## Getting Started

### Technologies Used
* React
* React Bootstrap
* Written in Typescript

### Installing
To set up:
* run `git clone https://github.com/YounusMZ/poetry-frontend.git`
* run `cd poetry-frontend` 
* and then run `npm install' to install the dependencies

### Executing app
To run:
* start the nodejs backend server
* run `npm run dev`

### Building app
To build:
* run `npm run build`

### Environment Variables
- `VITE_BACKEND_API_URL`: Set your API url
