# Poetry

## Description

Frontend for [poetry-backend](https://github.com/YounusMZ/poetry-backend). Provides a web-gui to search for poems imported from a file.

## Features
* Supports importing from csv(json planned).
* Shows random poems on the homepage. Explore every day.
* Made with react and react-bootstrap.
* Responsive UI for smaller screens.

## Getting Started

### Uses

* React
* react-bootstrap

### Installing
To set up:
* run `npm init`
* run `npm install react-bootstrap`

### Executing program
To run:
* start the nodejs backend
* run `npm run dev`
